# Report2_03 - Elective subject -

## 課題内容 - Contents  
* 標準入力(キーボード)から入力された非負整数を、オウム返しいそのまま表示するプログラムを作成せよ  
* 入力が非負である限り入出力を繰り返し、負数が入力されたらプログラムお終了うるようにせよ  
* レポートへの記入事項はRepore2_02と同様  

## C言語版プログラムのリスト
```c
#include <stdio.h>
#include <stdlib.h>

int main(){
	int a;
	while(1){
		printf(">");
		scanf("%d", &a);
		if(a < 0) break;
		printf("%d\n", a);
	}
	return 0;
}
```

## 完成したプログラムのリスト
```s
        .data                   # データセグメントにデータを配置することを示す
str:    .asciiz "\n"
msg:    .asciiz ">"
        .text                   # 以下の記述をテキストセグメントに配置
        .align  2               # 4byte境界になるように調整する
        .globl  main
main:
        addi    $t0, $0, 0      # t0 = 0を代入
loop:
        li      $v0, 4          # print_stringのシステムコールコードは4
        la      $a0, msg        # asciizで保存したプリントする文字列のアドレスを$a0に格納
        syscall                 # 文字列を表示
        li      $v0, 5          # 整数入力システムコールの番号５を設定
        syscall                 # 入力された整数が$v0に入る
        slt     $t0, $v0, $t0   # 入力された数($v0)が0より小さいか　小さい：$t0 = 1, 大きい：$t0 = 0
        bne     $t0, $0, out    # 入力された数が0より小さい場合はラベルoutへ
        move    $a0, $v0        # その整数をシステムコールの引数にする
        li      $v0, 1          # 整数表示システムコールの番号1を設
        syscall                 # 入力された整数が画面に表示される
        li      $v0, 4          # print_stringのシステムコールコードは4
        la      $a0, str        # asciizで保存したプリントする文字列のアドレスを$a0に格納
        syscall                 # 改行処理
        j       loop            # jump to loop
out:
        move    $v0, $0
        j       $ra             # jump to $ra

```

## プログラムの説明
```txt

ラベルmain内において、
入力された値が0より大きいか判断する為に使われるレジスタa0に0を格納する。
上記の動作の後、ラベルloopに入る。ラベルloop以下では、
1行目でsyscallでstring型を表示する為にレジスタv0に4を格納する。
2行目でレジスタa0に文字列msgの先頭アドレスを格納する。
3行目で文字列">"を表示する。
4行目でsyscallで整数型を読み込む為にレジスタv0に5を格納する。
5行目で入力された整数を読み込みv0に格納する。
6行目でレジスタv0に格納された値が0より小さいか(v0 <= 0)比較する。
小さいければレジスタt0に1を格納、大きければレジスタt0に0を格納する。
7行目でレジスタt0の値が0と等しいか比較し、等しくなければラベルoutのアドレスへ分岐する。
8行目で入力された値(a0)をsyscallの引数にする。
9行目でsyscallでint型を表示する為にレジスタv0に1を格納する。
10行目で入力された値が表示される。
11行目でsyscallでstring型を表示する為にレジスタv0に4を格納する。
12行目でレジスタa0に文字列strの先頭アドレスを格納する。
13行目で改行が行われる。
14行目で呼び出し側に戻る。
以上がラベルmain内の動作である。
ラベルout内の動作は、
1行目で返り値0をレジスタv0に設定する。
2行目で呼び出し側に戻る。

```

## 実行結果、および、その実行結果の正しさの説明
```txt
実行結果はC言語版プログラムも完成したプログラムも共に
0以上の値を入力した際はその数字が改行されて表示され、負の数が入力された際は表示されずにプログラムが終了した。
```
